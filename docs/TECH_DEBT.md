## 🔴 Незакрытая проблема: Ошибка 502 на Railway
- **Симптом:** При обращении к Railway (`https://bahus-production.up.railway.app/api/quotes`) Edge-сеть выкидывает статус 502 Bad Gateway ("Application failed to respond").
- **Что происходит внутри:** Контейнер успешно стартует и приложение на Python рапортует в логах, что слушает порт `0.0.0.0:8080`. Ошибок внутри Python-приложения нет.
- **Что было предпринято:** 
  1. Добавлен `railway.json` для принудительной сборки через Dockerfile (обход Nixpacks).
  2. Удалена инструкция `EXPOSE 8078` из `docker/postgres-api.Dockerfile` для исключения конфликта портов Railway Proxy.
